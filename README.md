# BU Scopus Research Dashboard

A WordPress-based research analytics dashboard developed for academic libraries to visualize institutional research output using Scopus, DSpace/Institutional Repository, Web of Science manual metrics, SDG mapping, subject analytics, country collaboration, and interactive publication details.

## Features

- Scopus publication analytics
- WordPress admin dashboard
- Public shortcode dashboard
- KPI cards
- Year-wise publication trend
- Citation analytics
- Open Access analysis
- Document type distribution
- Subject area visualization
- SDG goal mapping
- Country collaboration map
- DSpace/Institutional Repository thesis integration
- Manual Web of Science metrics
- AJAX-based popup details
- CSV export
- Cache-based dashboard loading
- WP-Cron based scheduled update

## Shortcode

Use this shortcode on any WordPress page:

```text
[bu_scopus_research_dashboard]
```

## API Key Notice

This repository does not include any API key, password, token, or private institutional credential.

After installing the plugin, authorized administrators can enter API credentials from:

WordPress Admin → BU Research Dashboard → Settings
